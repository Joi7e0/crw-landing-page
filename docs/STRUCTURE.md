# Структура репозиторію проєкту

Цей документ містить опис структури файлів та папок проєкту лендінгу бакалаврської роботи.

## Коренева директорія (Root)

*   `docs/` — Директорія з проектною документацією (тези, плани, звіти).
*   `public/` — Статичні ресурси, які доступні безпосередньо (логотипи, фавікони, robots.txt).
*   `src/` — Вихідний код React-застосунку.
*   `.gitignore` — Файл, що вказує Git, які файли/папки не потрібно відстежувати (наприклад, `node_modules`).
*   `index.html` — Основний HTML-файл, точка входу в браузер.
*   `LICENSE` — Ліцензія MIT, що визначає права на використання коду.
*   `package.json` — Файл з переліком залежностей та скриптів проєкту (npm).
*   `README.md` — Головний опис проєкту для GitHub.
*   `vite.config.js` — Конфігурація інструменту збірки Vite.

## Папка документів (`docs/`)

*   `task.md` — Список виконаних завдань та прогресу розробки.
*   `walkthrough.md` — Технічний опис функціоналу та результати верифікації.

## Папка вихідного коду (`src/`)

*   `components/` — React-компоненти, з яких складається сторінка:
    *   `Header.jsx` — Шапка сайту з навігацією та перемикачем тем.
    *   `Hero.jsx` — Головний банер із назвою роботи.
    *   `About.jsx` — Розділ про актуальність та вступ.
    *   `Goals.jsx` — Опис мети та функціональних вимог.
    *   `Methodology.jsx` — Архітектура та технологічний стек.
    *   `Results.jsx` — Результати та модулі системи.
    *   `Footer.jsx` — Футер із контактами та копірайтом.
    *   `Section.jsx` — Універсальна обгортка для секцій.
*   `App.jsx` — Головний компонент застосунку, який об'єднує всі частини.
*   `App.css` — Стилі лейауту, анімації та оформлення компонентів.
*   `index.css` — Глобальні стилі, змінні тем (Dark/Light) та скидання стилів.
*   `main.jsx` — Точка входу JavaScript, що рендерить React у DOM.
